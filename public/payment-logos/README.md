# Payment system logos

SVG-логотипы платёжных систем для рендера на лендинге и в блоке оплаты.

| Файл | Источник | Статус |
| --- | --- | --- |
| `visa.svg` | Visa Brand Center — Visa_Brandmark_Blue_RGB_2021 | Официальный |
| `mastercard.svg` | Mastercard Brand Center — `ma_symbol` | Официальный (только символ из двух кругов) |
| `mir.svg` | НСПК / mironline.ru | Официальный |
| `sbp.svg` | НСПК / sbp.nspk.ru | Официальный (полный лого с градиентом) |
| `tpay.svg` | brand.tbank.ru / личный кабинет | Официальный |
| `tbank.svg` | brand.tbank.ru | Официальный |

### Правила использования

- viewBox у файлов разный (от 60×40 до 3384×2077), `PaymentLogos.tsx` приводит их к одинаковой высоте через `h-6 w-auto`. Не нормализуем viewBox вручную, чтобы не повредить пропорции брендов.
- SVG-файлы не должны содержать `<script>` или внешних ссылок.
- Все изображения должны иметь `role="img"` или `<title>` для доступности (большинство уже имеют).
